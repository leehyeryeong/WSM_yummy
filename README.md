***YUMMY***
===========

**제철 과일을 알려주고, 그에 맞는 레시피를 알려주는 웹사이트입니다.**

-----------------------------------------------------

## 주요 효과

1. 잘 몰랐던 제철 과일들을 파악할 수 있습니다.
2. 어느 계절에는 어느 과일이 맛있는지를 알 수 있습니다.
3. 제철 과일이 왜 건강에 좋은지를 알려줍니다.
4. 과일로도 다양한 요리가 가능합니다.

---------------------------------------------------------------
## 주요 기능

- 계절별 카테고리를 볼 수 있습니다.
- 계절별로 제철 과일을 알 수 있습니다.
- 과일을 이용하여 할 수 있는 레시피들을 제공합니다.


|카테고리|내용|
|:---:|---|
|봄| 딸기, 한라봉, 매실
|여름| 수박, 포도, 복숭아 
|가을| 사과, 무화과, 석류
|겨울| 귤, 유자, 딸기
